Act as a Senior Principal Engineer. Review the code for Production Readiness with a strict focus on the following dimensions. For every issue found, provide the specific line number, the severity, and a refactored solution.

Review Checklist:

Security: Vulnerabilities (OWASP Top 10, Injection, Auth, PII/Data Privacy).
Performance & Scalability: Time complexity, N+1 queries, resource leaks, and concurrency handling.
Reliability: Error handling, crash resilience, and observability (logging/metrics).
Correctness: Logic errors, edge cases, race conditions, and business rule violations.
Quality Assurance: Test coverage gaps and test quality.
Maintainability: SOLID principles, design patterns, naming conventions, and modularity.
Documentation: API contracts, inline comments for complex logic, and README updates.
Architecture & Dependencies: System design alignment and 3rd-party library safety.



Scope: Only uncommitted changes in the Git working tree.
